## Introduction

This boilerplate uses [Playwright](https://playwright.dev/) for E2E testing.

## Installation

```bash
npx playwright install
```

## Running tests

1. Run development server

   ```bash
   npm run dev
   ```

1. Run Playwright

   ```bash
    npx playwright test --ui
    ```

    or

    ```bash
    npx playwright test
    ```
